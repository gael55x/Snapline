# Security policy

Snapline is currently pre-1.0. Security fixes target the latest published
release and the default branch. Once 1.x is published, the latest 1.x release
will receive security fixes; older pre-release versions are not supported.

Report vulnerabilities privately through this repository's GitHub security
advisory flow. Do not include exploit details, repository source, tokens, or
agent transcripts in a public issue.

Include the affected version, operating system and Node version, a minimal
reproduction, impact, and whether the issue crosses the project-root boundary
or exposes local source. Maintainers will acknowledge the report through the
same private channel before coordinating disclosure.
