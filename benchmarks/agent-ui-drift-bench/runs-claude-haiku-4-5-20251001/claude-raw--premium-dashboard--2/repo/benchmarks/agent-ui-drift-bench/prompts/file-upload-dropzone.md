Build a file upload component with a drag-and-drop zone that also opens a file picker on click. Show an icon and "Drag files here or click to browse" text, accept up to 5 files (max 10MB each, images and PDFs only). Below the zone, list each selected file with its name, size, a progress bar while uploading, and a remove button. Show an error state for rejected files.
