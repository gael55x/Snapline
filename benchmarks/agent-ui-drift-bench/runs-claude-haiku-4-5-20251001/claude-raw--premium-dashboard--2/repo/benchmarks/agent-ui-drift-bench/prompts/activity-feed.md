Add an activity feed to the dashboard sidebar showing the latest team events: someone created a project, commented, changed a setting, or invited a member. Each item shows the actor's avatar, a one-line description with the actor's name bolded, and a relative timestamp like "2h ago". Group items under Today / Yesterday / Earlier headers and add a "Load more" button at the bottom.
