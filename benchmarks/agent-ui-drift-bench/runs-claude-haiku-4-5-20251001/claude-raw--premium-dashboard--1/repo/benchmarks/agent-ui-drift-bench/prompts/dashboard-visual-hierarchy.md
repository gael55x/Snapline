Everything on the dashboard has the same visual weight, so nothing stands out. Rework the visual hierarchy: the key revenue number should dominate, secondary stats should recede, and the chart should read as the centerpiece below them. De-emphasize the sidebar widgets and make section headings clearly separate the zones. Don't add or remove any content.
