## New Components

### 1. **InviteMemberModal** (`src/components/invite-member-modal.tsx`)

A reusable modal component with:
- **Email input**: Accepts comma-separated email addresses
- **Role selector**: Dropdown with Admin and Member roles, each with descriptions:
  - Member: "Can view and contribute to projects"
  - Admin: "Full access to team settings and member management"
- **Personal message textarea**: Optional field for custom welcome messages
- **Seat management**: 
  - Displays available seats in the header description
  - Shows real-time count of entered emails
  - Displays remaining seats next to email count
  - Error message when exceeding seat limit
- **Submit button logic**:
  - Disabled when no emails entered or over seat limit
  - Shows "Sending..." state while processing
  - Transitions to success state after sending
- **Success state**: Shows a checkmark icon with confirmation message and invite count, auto-closes after 2 seconds
- **Form reset**: Clears all fields after successful submission
