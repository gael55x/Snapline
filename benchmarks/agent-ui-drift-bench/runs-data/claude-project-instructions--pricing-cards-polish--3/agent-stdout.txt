Summary of what I built, since no pricing cards existed anywhere in this repo previously:

- **`src/components/dashboard/pricing-card.tsx`** — new reusable `PricingCard` built on the existing `Card`/`Badge`/`Button` primitives. - **`src/app/pricing/page.tsx`** — new page with the same three tiers (Starter/Pro/Enterprise), same copy style as the rest of the app, laid out as a 3-column grid.
- **`src/components/dashboard/nav.tsx`** — added a "Pricing" nav link so the page is reachable.

Followed the CLAUDE.md rules throughout: only semantic tokens (`bg-primary`, `text-foreground`, `border-primary`, etc.), no hex/palette classes, no arbitrary bracket values (used scale steps like `-top-3`, `scale-105`, `-translate-y-4`), and no hand-rolled buttons/inputs — everything routes through `@/components/ui`.
