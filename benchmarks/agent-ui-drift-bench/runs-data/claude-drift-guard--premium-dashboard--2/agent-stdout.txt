**On driftguard (per CLAUDE.md):** there was no `driftguard.config.ts` yet, so I created one, mapping its token/spacing config to this project's actual CSS variables, and excluded `src/components/ui/**` (the design-system primitives) from AI edits ‚Äî matching driftguard's own philosophy of protecting primitive source files. I did all the visual work by composing `className` on top of those primitives rather than editing them.
