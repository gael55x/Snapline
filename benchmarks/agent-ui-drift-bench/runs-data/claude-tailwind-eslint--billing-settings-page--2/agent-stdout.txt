I've built the billing settings page. Here's a summary of what I hit and what I did:

**Page:** `src/app/settings/billing/page.tsx` — route `/settings/billing`.

**What's included:**
- Trial-ending banner (brand-colored) at the top, warning that the trial ends soon and the card will be charged.
- **Current plan** card: plan name, price/interval, trial badge, next billing date, and a "Cancel subscription" flow (confirmation dialog → shows a "your plan ends on X" notice and a "Resume subscription" button).
- **Payment method** card: card brand + last 4 digits, expiry, and an "Update card" button opening a form dialog (name, number, expiry, CVC) that updates the displayed card on save.
- **Invoices** card: last 3 invoices with date, description, amount, status badge, and a styled "Download" link per row.

All built from the existing primitives only (`Card`, `Button`, `Badge`, `Dialog`, `Input`), using only the theme's registered color tokens (`brand`, `border`, `muted`, `destructive`, etc.) — no arbitrary Tailwind values, no new dependencies.
