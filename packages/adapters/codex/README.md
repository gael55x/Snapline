# @usesnapline/codex

Codex project-hook payload parsing, response formatting, and reversible hook
installation for Snapline.

```sh
npm i @usesnapline/codex
```

This adapter is preview until a packed release candidate passes an interactive
Codex workflow. See the
[Codex integration guide](https://github.com/gael55x/Snapline/blob/main/docs/codex.md).
