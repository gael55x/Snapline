# @usesnapline/cursor

Cursor project-hook payload parsing, response formatting, and reversible hook
and rule installation for Snapline.

```sh
npm i @usesnapline/cursor
```

This adapter is preview until a packed release candidate passes an interactive
Cursor workflow. See the
[Cursor integration guide](https://github.com/gael55x/Snapline/blob/main/docs/cursor.md).
