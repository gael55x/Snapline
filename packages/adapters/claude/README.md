# @usesnapline/claude

Claude Code payload parsing, response formatting, and reversible project-hook
installation for Snapline.

```sh
npm i @usesnapline/claude
```

The supported user workflow is `snapline install claude`; adapter authors can
use this package directly. See the
[Claude integration guide](https://github.com/gael55x/Snapline/blob/main/docs/claude.md).
